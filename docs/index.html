Natrix UI is a professional-grade CSS framework that provides modern defaults, premium glassmorphism, and a robust utility system. It's Bootstrap for the modern era.

Quick Start

Copy this code into your head and you're ready to build.

<!-- Natrix CSS -->
<link rel="stylesheet" href="https://cdn.natrix.ui/dist/natrix.css">

<!-- Natrix JS (for Modals, Sidebar, etc) -->
<script src="https://cdn.natrix.ui/dist/natrix.js"></script>
